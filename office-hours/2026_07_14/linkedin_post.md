🐍 This week's Python + AI Office Hours covered GPT-5.6, agent harnesses, and lots of audience questions!

Topics we covered:

🤖 GPT-5.6 first impressions: Switched from Opus to Sol medium — cheaper, more conservative, needs leaner prompts. Model guidance recommends stating each instruction once and defining autonomy boundaries.

🔧 Agent Framework harness: Microsoft's batteries-included harness now ships compaction, todos, memory, background agents, looping, and OpenTelemetry out of the box.

📝 Markdown-first agents: Both GitHub Agentic Workflows and Azure Functions serverless agents let you define agents in Markdown with YAML constraints — no full framework needed.

🏗️ Planning with expensive models: Use Sol to create a plan.md or AGENTS.md, then hand off to a cheaper model in a fresh session. ADRs work great as durable handoffs too.

🔭 Observability: Use OpenTelemetry with GenAI semantic conventions for portability across Langfuse, App Insights, and Foundry tracing.

📊 Knowledge graphs: Azure AI Search is investigating GraphRAG again but nothing to announce yet. Google's Open Knowledge Format is being evaluated.

🚀 Hosted agents GA: Confirmed live by the PM — Foundry hosted agents are now generally available!
